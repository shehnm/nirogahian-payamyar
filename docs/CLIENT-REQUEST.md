# درخواست کارفرما — پونیشا / mosayebi

> متن استخراج‌شده از مکالمه پروژه. برای رمزها: `docs/ACCESS.md` (لوکال).

---

## عنوان پروژه (پونیشا)

**عدم نمایش دسته‌بندی‌ها و عدم ارسال خودکار به شبکه‌های اجتماعی** — سایت نیروگاهیان

---

## شرح مشکل اولیه (گزارش کارفرما)

1. در تنظیمات پیام‌یار (ایتا، بله، گپ، روبیکا) **دسته‌بندی‌ها نمایش داده نمی‌شد** یا خالی بود
2. **ارسال خودکار** پس از انتشار پست به کانال‌ها انجام نمی‌شد
3. گاهی پس از publish: «یک خطای مهم در این وب‌سایت وجود دارد»
4. ارسال **دستی** (دکمه خاکستری کنار پست‌ها) بعد از پچ اولیه برای برخی دسته‌ها کار کرد

### نمونه تست کارفرما

- انتشار خبر در دسته «اخبار آب» — خودکار نرفت، با دکمه خاکستری رفت
- فقط ۳ دسته در لیست: اخبار آب، سدها، فاضلاب — «اخبار» کلی نبود
- روبیکا جداگانه مشکل تنظیمات داشت (~۱۰۰ دسته بعد از پچ)

---

## انتظار صریح کارفرما

| # | انتظار |
|---|--------|
| 1 | **تمام** دسته‌بندی‌ها برای **تمام** پیام‌رسان‌ها در تنظیمات دیده شود |
| 2 | امکان تیک «اخبار» و سایر دسته‌های لازم |
| 3 | ارسال **خودکار** با انتشار پست — بدون دکمه دستی |
| 4 | سایت پایدار بماند (اولویت بالا — نه down) |
| 5 | فازبندی شفاف — نه پرداخت بی‌پایان |

### سوالات تکراری کارفرما

- «با ۵ میلیون مشکل ۱۰۰٪ اوکی میشه؟» — انتظار بسته شدن کامل فاز ۲
- «مجدد نیاز به فاز و بچ جدیده؟» — نمی‌خواهد فاز ۳ بی‌پایان
- «سایت را روی هاست دیگر / لوکال تست کنید» — برای دور زدن محدودیت دسترسی
- «یه سری دسته غیرضروری را حذف کنم؟» — آماده همکاری اگر لازم باشد

---

## توافق فازبندی (فریلنسر / کارفرما)

### فاز ۱ (~۹۰۰٬۰۰۰ تومان)

- سایت و پیام‌یار بالا
- روبیکا نسبتاً OK
- ایتا حدود ۳ دسته
- ارسال دستی برای تیک‌خورده‌ها
- **لیست کامل ۷۰۰ دسته + ارسال خودکار -> فاز ۲**

### فاز ۲ (۵٬۰۰۰٬۰۰۰ تومان — پچ جدید پونیشا)

- نمایش کامل دسته‌ها در UI (نه فقط cache بزرگ‌تر)
- اصلاح افزونه/پچ برای scale سایت
- ارسال خودکار «اخبار» و بقیه
- تست همه پیام‌رسان‌ها

---

## محدودیت‌های گزارش‌شده توسط کارفرما

- دسترسی FTP حتی با اکانت کامل گاهی مشکل دارد
- از UI (DirectAdmin / wp-admin) کندتر است
- سایت از IP خارج ایران timeout (نت‌افراز / محدودیت شبکه)
- SSH روی هاست اشتراکی نیست
- چند سایت روی همان اکانت `mosayebi` (سایت هدف: **nirogahian.ir** نه mosayebi-m.ir)

---

## نکات فنی از کارفرما

- تصاویر با افزونه جدا روی **هاست دانلود** بارگذاری و از آنجا خوانده می‌شود
- افزونه **تلگرام** روی سایت دیگر **همین هاست** کار می‌کند ولی روی nirogahian.ir نه — مشکل سایت‌محور
- توضیحات بیشتر در **منوی فوتر** سایت
- بکاپ DB: نت‌افراز ۱۹ خرداد + بکاپ دستی کارفرما روی هاست و فلش
- کانال تلگرام نمونه: `@nirogahian`

## معیار تحویل نهایی (قابل قبول کارفرما)

```
وقتی خبر با دسته اخبار (یا هر دسته) منتشر می‌شود
  -> خودکار به ایتا، بله، گپ، روبیکا برود
  -> در تنظیمات همه دسته‌ها دیده و قابل تیک باشند
  -> بدون خطای critical
  -> سایت up
```
